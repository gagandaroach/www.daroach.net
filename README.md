## Technology Stack

**Frontend**

* Vue 3
* Nuxt 3
* TailwindCSS

**Backend**

* Docker
* Archlinux

## Developer Guide

### System Setup

```sh
# Install Node Version Manager
yay nvm

# Install nodejs stable
nvm install --lts
```

### Running Dev Server

```sh
# Enter the web root directory
cd www

# Install nodejs packages
npm install

# Run dev server on local port
npm run dev
```
